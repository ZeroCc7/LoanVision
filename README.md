# LoanVision - 贷款与还款管理系统

LoanVision 是一款简洁、高效的个人贷款与还款管理工具。它专门针对“先息后本”等复杂还款方式进行了深度优化，帮助用户清晰地掌握资金流动趋势，预警大额还款压力。

## 🌟 核心功能

- **仪表盘统计**：直观展示本月总应还、已还、待还金额，以及总剩余待还本金。
- **还款趋势分析**：
  - 支持“过去 12 个月”历史回顾与“未来 12 个月”预测。
  - 采用组合图表（柱状图 + 折线图）清晰区分不同还款方式。
  - **先息后本专项优化**：自动识别并高亮标注本金到期月份，提供红色预警。
- **智能明细可视化**：
  - **双层饼图**：内层按还款方式（先息后本、等额类、信用卡）聚合，外层展示具体账目占比。
  - **明细列表**：支持按还款方式筛选，高亮显示先息后本账目，并拆解本金与利息。
- **到期预警系统**：针对未来 30 天内到期的先息后本贷款，在首页置顶显示红色专项预警。
- **全方位适配**：完美支持 **深色模式**，图表与界面均经过精心调优。

## 🚀 快速开始

### 1. 环境准备
确保您的机器已安装 [Node.js](https://nodejs.org/) (建议 v16+)。

### 2. 安装依赖
在项目根目录下运行：
```bash
npm install
cd client && npm install
cd ../server && npm install
```

### 3. 创建用户
由于系统采用 JWT 认证，您需要先创建一个用户：
```bash
npm run create-user <用户名> <密码>
```

### 4. 启动项目
运行以下命令同时启动前端和后端：
- **开发模式**：
  ```bash
  npm run client  # 启动前端 (http://localhost:5173)
  npm run server  # 启动后端 (http://localhost:5000)
  ```
- **生产模式**：
  ```bash
  npm run prod
  ```

## 🛠️ 技术栈

- **前端**：React + TypeScript + Ant Design + Recharts
- **后端**：Node.js + Express + SQLite (Better-SQLite3)
- **认证**：JWT (JSON Web Token)
- **部署**：支持 PM2 部署 ( ecosystem.config.js )
